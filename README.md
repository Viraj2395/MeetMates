# MeetMates
A full-stack event hosting and discovery platform built with Next.js, Convex, and Clerk.  
Focus areas include smooth onboarding, event discovery, event creation, and QR-based check-ins.

## Features
- Onboarding flow for new users
- Explore and search events
- Create and manage events
- QR-based check-in experience

## Tech Stack
- Next.js 
- React
- Tailwind CSS
- Radix UI / shadcn-style components
- Convex (backend + realtime data)
- Clerk (auth)

## Prerequisites
- Node.js 18+ 
- npm
